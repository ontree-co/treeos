Environment Variables

Domain settings can also be configured via environment variables, which take priority over database settings:

  • PUBLIC_BASE_DOMAIN - Sets the public base domain
  • TAILSCALE_BASE_DOMAIN - Sets the Tailscale base domain
Help
Setting up domains:
  1. Configure your DNS to point *.yourdomain.com to this server
  2. Enter your domain in the Public Base Domain field
  3. Caddy will automatically obtain HTTPS certificates
  4. Apps can then be exposed via the app detail page
Requirements:
  • Linux server (not macOS)
  • Caddy installed and running
  • Port 80 and 443 accessible
  • Valid DNS configuration
