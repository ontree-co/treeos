Environment Variables

Domain settings can also be configured via environment variables, which take priority over database settings:

  • PUBLIC_BASE_DOMAIN - Sets the public base domain
  • TAILSCALE_AUTH_KEY - Sets the Tailscale auth key
  • TAILSCALE_TAGS - Sets the Tailscale tags for apps
