# Dashboard and System Vitals

This document describes the main dashboard and system vitals monitoring features of the onTree Node application.

## Dashboard

The dashboard is the main landing page of the application after you have logged in. It provides a high-level overview of your system and applications.

## System Vitals

The dashboard displays real-time system vitals, including:

*   **CPU Usage:** The current CPU usage of the system.
*   **Memory Usage:** The current memory usage of the system.
*   **Disk Usage:** The current disk usage of the system.
*   **Network Usage:** The current network usage of the system.

You can click on each of these vitals to see a more detailed view.
